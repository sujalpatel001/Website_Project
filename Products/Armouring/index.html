Armouring Machine Specification


↠ MACHINE TYPE : Heavy duty Rigid Cradle, Disc and Planetary Type

↠ CAGES : 24, 28, 32, 36, 42, 48, 54, 36+36=72, 42+42=84, 48+48=96

↠ BOBBIN SIZE : 630mm / 500mm DIN Standard or as per customer's requirement

↠ CAGE RPM : 40 to 140 rpm.

↠ BOBBIN HOLDING : Shaftless by Pintles - H Type

↠ CAGE DIRECTION : Clock wise, anti clockwise, opposite to each other & same direction by lever operated enclosed gear boxes or electrically by PLC and HMI.

↠ MACHINE BRAKE : All cage will have pneumatic operated disc brake with solenoid valve & air regulator.

↠ DIE HOLDER : Separate die holders (Heavy Duty) for individual cages adjustable type with hand wheel and split type easy operation with hinges.
